# 🏋️ FORTIS – Демонстрация бэкенда для сбора заявок

Этот проект представляет собой **работающий сервер (бэкенд)**, который принимает заявки с веб-формы, сохраняет их в облачную базу данных и возвращает ответ.  
Создан как пример моего портфолио, чтобы Вы могли наглядно оценить качество разработки серверной части.

## 🔧 Что умеет этот бэкенд?

- Принимает заявки с формы (имя, email, тариф, сообщение).
- Проверяет корректность данных (например, тариф должен быть одним из допустимых значений).
- Сохраняет заявки в надёжную облачную базу данных MongoDB.
- Возвращает идентификатор созданной заявки – Вы всегда можете отследить каждое обращение.
- Отвечает на проверочные запросы – по адресу `/ping` сервер подтверждает, что работает.
- Документирует все свои возможности через Swagger – это как инструкция по эксплуатации для разработчиков.

## 🧪 Как проверить работу бэкенда без специальных знаний?

### 1. Пинг – самый быстрый тест
Перейдите по ссылке:  
[https://fortis-dev.onrender.com/ping](https://fortis-dev.onrender.com/ping)  
Вы увидите ответ:
```json
{ "status": "ok", "timestamp": 1234567890 }
Это значит, сервер запущен и готов к работе.

### 2. Виджет на сайте
Откройте демо-страницу:  
[https://dastfantast4-sys.github.io/fortis-dev/](https://dastfantast4-sys.github.io/fortis-dev/)  
В правом нижнем углу есть блок **«API Tester»**:
- Нажмите **«Ping»** – увидите ответ сервера.
- Введите любые тестовые данные (имя, email, тариф) и нажмите **«Отправить»** – заявка мгновенно уйдёт на сервер, сохранится в базу, а в логе появится подтверждение с уникальным номером.

### 3. Swagger – для более детального изучения
Перейдите по адресу:  
[https://fortis-dev.onrender.com/api-docs](https://fortis-dev.onrender.com/api-docs)  
Там вы сможете посмотреть все доступные методы API, их параметры и даже отправить тестовый запрос прямо из браузера.

## 📊 Технологии, которые использованы

| Компонент | Технология |
|-----------|------------|
| Серверная платформа | Node.js + Express |
| База данных | MongoDB (Mongoose) |
| Аутентификация | JWT, bcrypt |
| Валидация данных | Joi |
| Документация API | Swagger / OpenAPI |
| Хостинг бэкенда | Render |
| Управление версиями | Git, GitHub |

Все технологии выбраны с учётом надёжности, производительности и лёгкости масштабирования.
